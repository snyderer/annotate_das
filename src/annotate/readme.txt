To install:
In VSCode terminal, create a virtual environment using python 3.12:
    py -3.12 -m venv .venv

If you don't have python 3.12, download it here: https://www.python.org/downloads/release/python-3128/

Activate your virtual environment:
    windows: .\.venv\Scripts\Activate.ps1

Install with pip:
    pip install https://github.com/DAS4Whales/Annotate.git
